### Dev

先安装依赖包
```nodejs
npm install
```

开启代理
```nodejs
npm run server
```

webpack-server
```nodejs
npm run dev
```

[http://localhost:8080](http://localhost:8080)

有时候图会挂掉，因为知乎做了防外链，走的代理，不太稳定
