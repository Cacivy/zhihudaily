### Dev

先安装依赖包
```nodejs
npm install
```

安装node-sass需要使用cnpm

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install node-sass
```

开启代理
```nodejs
npm run server
```

webpack-server
```nodejs
npm run dev
```

[http://localhost:8080](http://localhost:8080)
